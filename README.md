# Medicine Stock & Expiry Tracking System (SQL)

## Description
A DBMS mini project that helps clinics track medicine availability,
identify low stock, and monitor expiry dates using SQL queries.

## Technologies Used
- SQL
- SQLite / Oracle SQL

## Features
- Tracks out-of-stock medicines
- Identifies medicines nearing expiry
- Calculates total medicine availability per clinic
- Uses JOINs and aggregate functions

## Use Case
Helps clinics reduce medicine wastage and avoid shortages by maintaining
proper medicine inventory.
